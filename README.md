# ScratchGame
A scratch game, that will generate a matrix (for example 3x3) from symbols(based on probabilities for each individual cell) and based on winning combintations user either will win or lost.
User will place a bet with any amount which we call *betting amount* in this assignment.
.jar can be run through CLI commad
java -jar game.jar --config config.json --betting-amount 100  
where config.json should be a path to config, for example,
java -jar game.jar --config C:\Users\user\Desktop\config.json --betting-amount 100
